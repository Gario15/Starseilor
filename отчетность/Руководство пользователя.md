## 📖 Описание

**Starseilor** — это аддон для Blender, предназначенный для автоматического размещения объектов вдоль кривых. Инструмент позволяет быстро создавать повторяющиеся элементы, следующие по заданной траектории, и идеально подходит для создания:

- заборов и ограждений
- гирлянд, цепочек и украшений
- дорожных столбов, опор освещения
- архитектурных элементов (колонны, балки, перила)
- ландшафтных объектов (деревья, камни вдоль дорожек)
- любых повторяющихся элементов вдоль пути

---

## ✨ Основные возможности

### Режимы размещения
- **Point Objects** — размещение объектов в контрольных точках кривой
- **Edge Objects** — размещение объектов вдоль сегментов между точками

### Управление плотностью
- Размещение по фиксированному расстоянию между объектами
- Размещение по точному количеству объектов на каждом сегменте
- Настройка отступов от начала и конца сегментов

### Контроль области
- Ограничение по диапазону кривой
- Отсечение объектов от концов кривой на заданное расстояние

### Трансформации объектов
- Равномерный и раздельный масштаб по осям
- Точная настройка поворота в радианах
- Случайные вариации масштаба и поворота с контролируемым seed

### Управление кривой
- Подразделение кривой для более точного размещения на изгибах
- Быстрое переключение типа ручек

### Организация в сцене
- Автоматическое создание отдельных коллекций для каждой кривой
- Имена коллекций в формате `Starseilor_<имя_кривой>`

### Производительность
- Оптимизированное пакетное создание объектов
- Кэширование случайных значений
- Мгновенное обновление при изменении параметров

## 🔧 Установка

### Способ 1: Через интерфейс Blender

1. Сохраните код аддона в файл с расширением `.py`
2. Откройте Blender
3. Перейдите в меню: **Edit → Preferences**
4. Выберите вкладку **Add-ons**
5. Нажмите кнопку **Install...** в правом верхнем углу
6. Найдите сохраненный файл `starseilor.py` и выберите его
7. Включите аддон, поставив галочку рядом с **"3D View: Starseilor"**

### Способ 2: Ручная установка

1. Скопируйте файл `starseilor.py` в папку аддонов Blender:
   - **Windows:** `C:\Users\<Имя>\AppData\Roaming\Blender Foundation\Blender\4.0\scripts\addons\`
   - **macOS:** `~/Library/Application Support/Blender/4.0/scripts/addons/`
   - **Linux:** `~/.config/blender/4.0/scripts/addons/`
2. Перезапустите Blender
3. Включите аддон в **Edit → Preferences → Add-ons**

### Проверка установки

После включения аддона в правой боковой панели 3D Viewport должна появиться вкладка **Starseilor**.

## 🚀 Быстрый старт

### За 5 шагов к результату

**Шаг 1: Подготовьте сцену**
- Создайте объект для размещения: `Shift+A → Mesh → Cube`
- Создайте кривую: `Shift+A → Curve → Bezier`

**Шаг 2: Настройте кривую**
- Выделите кривую, перейдите в Edit Mode
- Придайте кривой желаемую форму, двигая контрольные точки
- Вернитесь в Object Mode

**Шаг 3: Откройте аддон**
- Нажмите `N` для открытия боковой панели
- Перейдите на вкладку **Starseilor**

**Шаг 4: Назначьте объекты**
- В поле **Point Object** выберите куб
- Поле **Edge Object** можно оставить пустым

**Шаг 5: Разместите объекты**
- Нажмите кнопку **Place Objects**
- Объекты появятся вдоль кривой в коллекции `starseilor_BezierCurve`

### Первые эксперименты

- **Больше объектов:** увеличьте `Subdivide` в `Curve Options`
- **Расстояние между объектами:** включите `Use Distance` в `Edge Spacing` и установите значение
- **Случайный масштаб:** в `Scale Settings` включите `Random Scale`
- **Очистить:** кнопка `Clear` удаляет объекты текущей кривой

## ⌨️ Горячие клавиши и советы

| Действие | Сочетание |
|----------|-----------|
| Открыть боковую панель | `N` |
| Редактировать кривую | `Tab` |
| Обновить размещение | Кнопка **Place Objects** |

### Советы по использованию

- **Создайте несколько кривых** — каждая получит свою коллекцию, и вы сможете управлять ими независимо
- **Используйте одинаковый Seed** для воспроизводимых результатов рандомизации при тестировании
- **После изменения формы кривой** в Edit Mode обязательно нажмите **Place Objects** снова
- **Комбинируйте Point и Edge объекты** для создания сложных конструкций
- **Экспериментируйте с Subdivide** — одно нажатие может кардинально изменить результат на изогнутых участках
- **Сохраняйте пресеты** — настройки сохраняются в blend-файле, создайте библиотеку часто используемых конфигураций

---

## ⚠️ Известные ограничения

1. При удалении объекта-шаблона созданные копии остаются в сцене
2. Offset Start + Offset End не должны превышать длину сегмента
3. Subdivide выше 3 может создать очень много сегментов на сложных кривых
4. Аддон работает только с активным объектом-кривой
